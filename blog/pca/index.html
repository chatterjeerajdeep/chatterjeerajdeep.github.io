What? Why? How?

Published on Aug 15, 2021

Reading time: 2 minutes.

What makes data interesting?

We always keep talking about data and how we belong to the age of data. But what is it that makes data so exciting? Is it about the volume or the variety of data getting generated? Is it about the different ways of applying all the maths and processing the data?

I believe, it is the uncertainty in data, and our quest to explain this uncertain behaviour which is at the core of everything interesting about data.

For example, a document that has a fixed set of frequently asked questions about any service seems boring. But the same information being provided by a bot, makes it interesting. This is because fundamentally, we love the idea of the uncertainty that a freely flowing conversation can generate.

The same thing applies for trends in the stock market or in the detection of diseases or maybe in detecting fraudulent transactions.

This uncertainty in data is called variance. Everything interesting about data are the different ways to explain this variance.

How can we explain the variance in data?

There can be different kinds of data that we can collect, but essentially it is either a set of observations or decisions.

Suppose, we are collecting data about the performance of a team in a game. There are three possible decisions - win, lose and tie. After 20 games, we will have a set of 20 observations, each being one of either win, lose or tie.

Now, if we want to explain these observations, we have to figure out the possible set of reasons that can contribute to the team’s win or loss. These reasons can be like the hours of practice, the number of experienced players, weather conditions, etc. The ineteresting part here is again the uncertainty. Some of these reasons may actually help in explaining the team’s performance whereas some of them may be completely irrelevant.

The observation data that we are trying the explain is called the target variable. The possible set of factors leading to the observations are known as features or feature variables.

The feature variables are supposed to explain the variance in target data.

Finding features

If we can find the correct set of features, we should be able to explain the patterns in the observed behaviour. This will enable us to predict the behaviour in future circumstances. That would be great in many many ways!

So obviously the next interesting thing is to find the relevant features. Pure guesses or assumptions may not always help. A subject matter expert or group of experts can suggest possible significant features. Sometimes, careful observations can help in finding relevant features.

Not all features contribute equally in explaining the target variable. For example, having more number of experienced players may be more important than the weather condition in determining the performance of the team.

In most practical use-cases, we have a given set of features and a target variable. We have to find which features are important in explaining the target variable and by how much.
